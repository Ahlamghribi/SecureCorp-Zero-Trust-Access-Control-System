Attack Scenarios

Security control validation against common attack vectors

Live Attack Lab — Replay Attack

3 étapes réelles contre le KDC (port 8000). Tu choisis toi-même : envoyer un nonce propre ou le rejouer.

01
Login → Obtenir un TGT
Appelle POST /login sur le KDC. Tu reçois un TGT signé.
02
Demande de ticket — choisis ton nonce
Entre un nonce manuellement. Envoie une première requête légitime. Puis tu peux choisir : changer le nonce (requête normale) ou garder le même (replay attack).
Attaque : Le nonce est déjà enregistré dans le KDC. Clique ci-dessous pour le renvoyer exactement pareil → le KDC doit rejeter avec 401.
Replay Attack détectée et bloquée par le KDC
Le serveur stocke chaque nonce dans USED_NONCES. Toute réutilisation → HTTP 401 immédiat.

Live Attack Lab — Ticket Tampering

Obtiens un vrai JWT du KDC. Envoie-le intact ou falsifié au Resource Server (port 8002).

01
Obtenir un ticket JWT légitime depuis le KDC
Appelle POST /login + POST /request-ticket. Tu obtiens un JWT signé HMAC-SHA256.
02
Envoie le ticket — intact ou falsifié — au Resource Server
Le ticket ci-dessous est le vrai JWT. Envoie-le intact (requête légitime) ou falsifié (attaque).
Falsification détectée — JWT invalide rejeté
La vérification HMAC-SHA256 échoue dès qu'un seul caractère est modifié. Le Resource Server retourne HTTP 401.

Live Attack Lab — Escalade de Privilèges

Connecte-toi avec un utilisateur limité, récupère un ticket réel, puis tente une action interdite sur le Resource Server.

01
Choisir l'utilisateur et l'action à tenter
Le ticket sera obtenu automatiquement via le KDC. L'action sera envoyée au Resource Server.
Valid Access
Alice (Admin) reads the public Company Handbook. Expected: ALLOW.
Insufficient Clearance
Carol (public clearance) requests a secret Finance document. Expected: DENY.
External Location Blocked
Dave (external) requests a secret resource. ABAC policy-1 blocks. Expected: DENY.
Replay Attack
Nonce is reused to replay a ticket request. KDC nonce store blocks it. Expected: 401.
Ticket Tampering
JWT signature is corrupted before presenting to the resource server. Expected: 401.
Privilege Escalation
Carol (Employee) attempts a DELETE operation. RBAC blocks. Expected: 403.
Invalid Credentials
Login attempt with wrong password. KDC rejects. Expected: 401.
Authorized Manager Access
Bob (Manager/Finance) reads a Finance secret document. Expected: ALLOW or DENY per clearance.
